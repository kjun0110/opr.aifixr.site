AIFIX – 원청사 데이터 조회/관리 탭 구조 통합 (직무 기반 권한 구조)

너는 제조업 공급망 ESG/PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 원청사 화면에는 다음 두 개의 탭이 존재한다.

데이터 조회

데이터 관리

하지만 현재 구조는 업무 흐름 측면에서 비효율적이다.

사용자는 데이터를 조회한 뒤 수정하려면

데이터 조회 → 데이터 관리 탭 이동 → 동일 조건 다시 조회 → 수정

이라는 비효율적인 과정을 거쳐야 한다.

따라서 데이터 조회와 데이터 관리 기능을 하나의 탭으로 통합하여 UX를 개선하라.

1️⃣ 탭 구조 변경

데이터 관리

변경

데이터 관리 (조회 + 수정 통합)

즉 데이터 조회 탭을 데이터 관리 탭에 흡수한다.

⚠️ 중요

현재 드롭다운 구성과 선택 옵션은 변경하지 말고 그대로 유지한다.

이 조회 조건은 데이터 조회와 데이터 관리 모두의 기준 필터로 사용된다.
