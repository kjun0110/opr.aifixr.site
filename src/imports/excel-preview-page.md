AIFIX – 공급망 통합 Excel 다운로드 미리보기 페이지 설계 (모달 → 페이지 변경)

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 데이터 관리 탭의 조회 결과 카드 헤더 우측에는 다음 버튼이 존재한다.

[공급망 보기] [엑셀 다운로드] [엑셀 업로드]

현재 구현에서는 엑셀 다운로드 버튼 클릭 시 모달이 열리도록 되어 있다.

그러나 이는 요구사항과 다르다.

엑셀 다운로드 클릭 시 모달이 아니라 별도 페이지로 이동하도록 UI 구조를 수정하라.

1. 버튼 동작 수정

현재

엑셀 다운로드 → Excel Preview Modal

수정

엑셀 다운로드 → Excel Preview Page

즉 모달을 제거하고 별도 페이지(depth page)로 이동하도록 변경한다.

2. 페이지 이동 구조

라우팅 구조

데이터 관리
 > 공급망 조회
 > Excel 다운로드 미리보기

Breadcrumb 추가

데이터 관리 > 조회 결과 > Excel 다운로드 미리보기
3. 페이지 목적

이 페이지는 실제 Excel 다운로드 기능이 아니라

해당 공급망에서 다운로드될 Excel 파일의 구조를 미리 보여주는 화면

이다.

즉 Excel 파일의 Workbook 구조를 UI로 시각화하는 페이지이다.

6. Excel Workbook 미리보기 UI

페이지 본문은 Excel UI를 모방한 구조로 설계한다.

구조

좌측 : Sheet Navigator
우측 : Sheet Table Preview
7. Sheet Navigator

좌측 패널에 Excel 시트 목록을 표시한다.

시트 목록

1 공급망 구조
2 기업정보
3 사업장
4 납품제품
5 자재/품목
6 생산투입실적
7 공정운영
8 담당자

시트 선택 시 우측 테이블이 변경된다.

8. Sheet Preview Table

각 시트는 전체 데이터가 아니라 샘플 row 3~5개 정도만 표시한다.

테이블 상단에는 다음 정보를 표시한다.

시트명
설명
포함 대상
컬럼 수

예시

시트명: 기업 및 조직 식별 정보
설명: 공급망 내 모든 회사의 공통 식별 정보
포함 대상: 원청 + 협력사
컬럼 수: 16
9. Excel 시트 구성

이 Excel은 원청 + 협력사 데이터를 하나의 파일로 통합한 구조이다.

그러나 스키마 차이 때문에 시트를 분리한다.

시트 목록

Sheet 1 공급망 구조
Sheet 2 기업 및 조직 식별 정보
Sheet 3 사업장 정보
Sheet 4 납품 제품 정보 (협력사)
Sheet 5 자재 및 품목 (원청)
Sheet 6 생산 및 투입 실적 (협력사)
Sheet 7 공정 운영 및 정밀 실적 (원청)
Sheet 8 담당자 정보
10. 기존 상세 페이지와의 관계

기존 기능은 유지한다.

상세보기 → 회사 단건 데이터 확인
엑셀 다운로드 → 공급망 전체 Excel 미리보기

즉

상세보기 = 회사 단건
엑셀 다운로드 = 공급망 전체

기능 목적을 명확히 분리한다.

11. 엑셀 업로드 버튼

공급망 카드 헤더의 엑셀 업로드 버튼은

향후

데이터 수정 업로드

기능으로 연결될 수 있도록 UI 버튼만 유지한다.

12. 중요한 UI 원칙

이 페이지는

"다운로드될 Excel 파일의 구조를 사용자에게 설명하는 화면"

이다.

따라서 반드시 다음이 보이도록 설계한다.

공급망 전체 데이터가 하나의 Excel 파일로 묶임

원청과 협력 데이터가 함께 포함됨

스키마 차이 때문에 시트는 분리됨

Excel workbook 구조가 직관적으로 이해 가능함
