따라서 조회 결과 테이블의 가로 레이아웃을 개선하라.

1️⃣ 테이블 가로 스크롤 적용

협력사 데이터 테이블에 가로 스크롤(horizontal scroll)을 적용한다.

구조

프로젝트 카드
→ 펼치기
→ 협력사 데이터 테이블 (가로 스크롤 가능)

즉

테이블이 컨테이너 width를 초과할 경우
오른쪽으로 스크롤하여 모든 컬럼을 볼 수 있도록 한다.

2️⃣ 핵심 컬럼 Sticky 처리

사용자가 가로 스크롤을 하더라도
공급망 맥락을 잃지 않도록 왼쪽 핵심 컬럼을 고정(sticky column)한다.
