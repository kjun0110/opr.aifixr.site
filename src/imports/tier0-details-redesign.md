AIFIX – Tier0(원청사) 상세보기 화면 재설계

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 데이터 관리 화면에서 Tier0 회사(예: 삼성SDI) 의 상세보기 버튼을 누르면 협력사 화면(UI)이 열리는 오류가 있다.

Tier0는 원청사(Operator) 이므로 협력사와 데이터 구조와 입력 방식이 다르다.

따라서 Tier0 전용 상세보기 화면을 새로 설계하라.

1. 화면 진입 구조

데이터 관리 → 공급망 리스트 → 회사 Row → 상세보기

조건 분기

Tier0 = 원청사 화면
Tier1~TierN = 협력사 화면

현재 협력사 화면을 그대로 쓰지 말고
Tier0 전용 상세 화면을 별도 설계한다.

2. 화면 상단 회사 정보 카드
