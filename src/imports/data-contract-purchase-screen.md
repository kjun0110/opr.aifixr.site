0) 화면 목적 (Goal)

[DATA CONTRACT]는 법무/관리 부서가 확정한 제3자 개인정보 이용 동의서(이하 동의서) 및
협력사 회원가입 시작 시(개인정보 입력 전) 진행되는 실제 동의 절차/동의 항목을
버전 관리 + 현황 관리 + PDF 관리하는 공간이다.

구매직무는:

“현재 적용 중인 동의서 버전”을 확인하고

초대 탭에서 드롭다운으로 선택할 버전을 확인하며

새 버전 적용/검토를 “요청”할 수 있어야 한다.

단, 구매직무는 확정된 동의서 PDF를 직접 수정할 수 없다(읽기 전용 + 요청 기반).

1) 페이지 레이아웃(1440px)

하단 또는 우측: 동의 항목(Consent Items) + 적용 현황(Status) 탭 구조
