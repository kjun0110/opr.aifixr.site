AIFIX – 공정 데이터 구조 개선 (Process Definition / Activity 분리)

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 Tier0(원청사) 상세 화면에서
공정 관련 데이터 구조가 다음 두 탭으로 구성되어 있다.

현재 구조

공정 운영 정보 (설비)
공정 운영 및 정밀 실적 정보

이 구조는 LCA / PCF 데이터 모델 기준에서 혼동을 유발한다.

공정 데이터는 다음 두 가지로 명확히 분리되어야 한다.

Process Definition (공정 정의)
Process Activity Data (공정 활동 데이터)
