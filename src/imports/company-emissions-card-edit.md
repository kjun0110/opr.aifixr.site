너는 제조업 ESG/PCF 관리 SaaS **AIFIX**의 시니어 프로덕트 디자이너다.

현재 “데이터 관리 > 자사 데이터 관리” 화면의
**자사 배출량 데이터 카드 UI를 수정하라.**

현재 테이블의 마지막 열에 **연필 아이콘 형태의 수정 버튼이 존재**하며
각 행(Row)을 개별적으로 수정하는 구조로 되어 있다.

그러나 이 방식은 다음 문제가 있다.

* 수정 아이콘이 작아 가시성이 낮다
* 행 단위 수정 구조가 직관적이지 않다
* 데이터 관리 화면에서는 **카드 단위 편집 모드**가 더 자연스럽다

따라서 아래 기준에 따라 UI를 수정하라.

---

### 1. "수정" 열 삭제

테이블의 마지막 열에 있는

* "수정" 컬럼
* 연필 아이콘

을 **완전히 제거한다.**

---

### 2. 카드 단위 Edit Mode 버튼 추가

"자사 배출량 데이터" 카드 상단의 버튼 영역에
**편집 모드 전환 버튼을 추가한다.**

현재 버튼 구조

엑셀 다운로드
엑셀 업로드
저장

이를 다음 구조로 변경한다.

엑셀 다운로드
엑셀 업로드
수정
저장

수정 버튼은 **저장 버튼의 왼쪽**에 배치한다.

---

### 3. Edit Mode 전환 동작

사용자가 **수정 버튼을 클릭하면 카드 전체가 Edit Mode로 전환**된다.

Edit Mode 상태에서는

* 테이블의 입력 값들이 수정 가능한 Input 형태로 변경된 것처럼 보인다
* 테이블 배경이 약하게 강조된다

---

### 4. Edit Mode 버튼 변경

편집 모드 상태에서는 버튼이 다음과 같이 변경된다.

엑셀 다운로드
엑셀 업로드
취소
저장

취소 버튼은 수정 버튼 위치에 표시된다.

---

### 5. 저장 동작

저장 버튼 클릭 시

* 수정된 데이터가 저장되고
* 카드가 다시 **읽기 모드(Read Mode)** 로 돌아간다.

---

### 6. 취소 동작

취소 버튼 클릭 시

* 수정된 값은 저장되지 않는다
* 카드가 읽기 모드로 돌아간다.

---

### 7. UX 목적

이 화면은 **자사 배출 데이터 관리 화면**이므로
사용자가 다음 흐름으로 작업할 수 있어야 한다.

데이터 조회
→ 카드 편집 모드 전환
→ 여러 행 데이터 수정
→ 저장

따라서 **행 단위 수정 UI 대신 카드 단위 Edit Mode UX를 적용한다.**
