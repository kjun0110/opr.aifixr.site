AIFIX – 협력사 초대 탭 직무 권한(R&R) 기반 UI 수정

너는 제조업 공급망 ESG/PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 “협력사 초대” 탭은 상단에서 구매 관점 / PCF 관점을 전환할 수 있으며
두 관점 모두 초대 기능을 수정할 수 있게 되어 있다.

하지만 실제 업무 R&R 기준에서는 다음과 같이 역할이 분리된다.
