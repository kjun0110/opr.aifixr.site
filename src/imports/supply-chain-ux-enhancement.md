AIFIX – 데이터 조회 화면 공급망 선택 UX 보강

현재 데이터 조회 화면에는
적용 공급망 표시 바가 구현되어 있다.

이를 다음과 같이 고도화하라.

1️⃣ 프로젝트 카드 내 매칭 방식 표시

각 SCN 카드 상단에 다음 배지 추가:

자동 매칭

수동 선택

위치는:

SCN-2026-01 옆, 버전 표시 근처.

목적:

사용자가 왜 해당 공급망이 적용됐는지 즉시 인지 가능하도록.

2️⃣ 상단 공급망 표시 바 정보 보강

현재:

적용 공급망: SC-1.2

다음 정보 추가:

매칭 방식: 자동 / 수동

선택 기준:
예: “프로젝트에 연결된 제품 기준”

구조 변경 여부 배지 표시

예시:

적용 공급망: SC-1.2
구조 변경: 없음
매칭 방식: 자동

3️⃣ “다른 버전 선택” 클릭 시 모달 고도화

단순 드롭다운이 아니라:

모달 구조:

좌측:

공급망 그룹 리스트

우측:

선택 그룹의 버전 타임라인

표시 예:

SC-1
1.0 (구조 생성)
1.1 (정보 수정)
1.2 (정보 수정)

SC-2
2.0 (구조 변경)

각 버전에:

생성일

변경 유형

변경 사유

협력사 수

구조 변경 여부 배지

4️⃣ 버전 비교 기능 (선택 사항)

버전 2개 선택 시:

“변경 내용 비교” 버튼 표시

예:

협력사 추가 1건

담당자 변경 2건

5️⃣ UX 원칙

사용자가 공급망을 직접 고를 필요는 없도록 유지

자동 매칭을 기본으로 하되,

수동 선택은 명확한 책임 UX로 표현

📌 최종 정리

지금 화면은 80% 완성.

매칭 투명성

버전 선택 모달 고도화

자동/수동 책임 구분
