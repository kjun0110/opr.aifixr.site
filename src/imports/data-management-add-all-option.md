AIFIX – 데이터 관리 탭 입력/관리 조건 드롭다운에 “ALL” 옵션 추가

너는 AIFIX의 시니어 프로덕트 디자이너다.
현재 [데이터 관리] 탭 > 입력/관리 조건 카드의 드롭다운 구성에 수정이 필요하다.

🎯 수정 목적

모든 드롭다운에 기본 선택지로 “ALL (전체)” 옵션을 추가하라.

데이터 조회 탭과 UX를 통일하고,
구매직무가 특정 조건을 제한하지 않고 전체 범위 기준으로 관리할 수 있도록 하기 위함이다.

1️⃣ 수정 대상 드롭다운 (모두 포함)

입력/관리 조건 카드 안에 있는 다음 항목들:

▽ 고객사

▽ 프로젝트

▽ 제품

▽ 사업장

▽ 공장

▽ 공정

▽ 기간 (시작 년월 / 종료 년월 → 기간은 제외, 아래 설명 참고)

2️⃣ ALL 옵션 추가 규칙
🔹 기본 규칙

각 드롭다운 최상단에:

ALL (전체)

옵션을 추가한다.

예시 구조

ALL (전체)

BMW

Mercedes

Tesla

🔹 기본 선택 상태

초기 로딩 시 기본값은:

고객사: ALL

프로젝트: ALL

제품: ALL

사업장: ALL

공장: ALL

공정: ALL

즉, 조건 미선택 상태가 아니라
“전체 범위 관리 모드”가 기본값이 되도록 한다.

3️⃣ 기간 드롭다운 처리 방식

현재 기간은:

시작 년월

종료 년월

두 개로 구성되어 있음.

기간은 ALL 개념 대신 다음 UX 적용:

미선택 상태 = 전체 기간

또는

“전체 기간” 체크박스 추가

권장안:

시작~종료 드롭다운 위에

☐ 전체 기간

체크 시:

두 드롭다운 비활성화

전체 기간 조회/관리

4️⃣ UI 스타일 유지

ALL은 일반 옵션과 동일 스타일

선택 시 선택 태그에 “ALL” 표시

여러 드롭다운이 ALL일 경우, 조회 결과는 전체 범위 기준

5️⃣ Empty 상태 문구도 수정

기존:

조건을 선택해주세요

수정:

조건을 선택하거나 기본값(ALL) 기준으로 조회/불러오기를 진행하세요.

6️⃣ 최종 목표

데이터 조회 탭과 UX 패턴 통일

구매직무가 조건 제한 없이 전체 관리 가능

엔터프라이즈 SaaS 수준의 일관된 필터 UX
