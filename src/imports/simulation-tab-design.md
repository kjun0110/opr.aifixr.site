0️⃣ 화면 목적 (Goal)

구매직무가 다음 질문에 답할 수 있도록 설계한다:

특정 공급망 변경 시 원가 영향은?

특정 원자재 가격 상승 시 손익 영향은?

특정 협력사 리스크 발생 시 대체 시나리오는?

특정 기간 내 생산량 변동 시 구매비용 영향은?

즉,
👉 공급망 기반 재무 영향 시뮬레이션 대시보드

1️⃣ 전체 레이아웃 구조

화면은 크게 3영역으로 구성한다:

시나리오 설정 패널 (좌측 또는 상단 카드)

시뮬레이션 결과 요약 KPI 영역

상세 분석 차트 및 비교 영역

데스크톱 1440px 기준 설계

2️⃣ 시나리오 설정 영역

카드 형태로 상단에 배치

🔹 기본 선택 필터

▽ 고객사

▽ 프로젝트

▽ 제품

▽ 공급망 버전

▽ 적용 기간

🔹 시나리오 변수 설정

다음 입력 필드를 포함하라:

① 원자재 가격 변동 (%)

슬라이더 + 숫자 입력

예: +5%, -3%

② 특정 협력사 제외 시뮬레이션

협력사 드롭다운

“대체 공급망 자동 재계산” 토글

③ 생산량 증감 (%)

슬라이더 + 숫자 입력

④ 환율 변동 (%)

선택 옵션 (고급 설정 접기/펼치기)

🔹 실행 버튼

[시뮬레이션 실행] (Primary Gradient 버튼)

로딩 시 애니메이션 표시
