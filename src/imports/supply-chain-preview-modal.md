AIFIX – 공급망 통합 Excel 다운로드 미리보기 모달 설계

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 데이터 관리 탭의 조회 결과 카드 헤더 우측에
공급망 보기 버튼이 있다.

이 버튼 옆에 아래 버튼 2개를 추가하라.

엑셀 다운로드

엑셀 업로드

중요:
피그마에서는 실제 Excel 파일 다운로드가 구현되지 않으므로,
엑셀 다운로드 버튼 클릭 시 실제 파일 다운로드가 아니라 “다운로드 미리보기 모달” 이 열리도록 설계하라.

1. 버튼 위치

공급망 카드 헤더 우측 액션 영역

SCN-2026-01 | BMW iX5 배터리 공급                  [공급망 보기] [엑셀 다운로드] [엑셀 업로드]
2. 엑셀 다운로드 클릭 시 동작

엑셀 다운로드 버튼 클릭 시
대형 모달(Large Modal / Full-width Dialog) 이 열리도록 설계한다.

이 모달은 실제 다운로드 실행창이 아니라
“이 공급망에 대해 다운로드될 Excel 파일의 구성 미리보기” 용도다.

모달 제목 예시:

공급망 통합 Excel 미리보기
SCN-2026-01 | BMW iX5 배터리 공급

우측 상단 액션:

닫기

다운로드 실행(비활성 또는 더미 버튼 가능)

4. 하단 본문 구조

본문은 Excel workbook 미리보기 UI로 설계한다.

형태:

좌측: 시트 목록 패널 또는 하단 시트 탭

우측: 선택된 시트의 표 미리보기

전체 스타일은 엑셀 화면처럼 보이게 한다

하단: 시트 탭

또는

둘 중 하나로 설계

탭 이름은 다음과 같이 짧게 표시한다.

탭 클릭 시 해당 시트 미리보기 테이블이 바뀌도록 설계한다.

7. 모달 안 테이블 표현 방식

실제 데이터 전체를 다 넣지 말고
각 시트마다 샘플 row 3~5개 정도만 보여주는 미리보기로 구성한다.

테이블 상단에는 아래 정보를 넣는다.

예시:

시트명: 기업 및 조직 식별 정보
설명: 공급망 내 모든 회사의 공통 식별 정보
포함 대상: 원청 + 협력사
8. 중요한 설계 원칙

이 미리보기 모달은
사용자에게 “어떤 Excel이 다운로드될지”를 설명하는 화면이다.

따라서 반드시 아래를 만족해야 한다.

원청과 협력 정보가 하나의 파일로 묶인다는 점이 보여야 함

하지만 스키마 차이 때문에 시트는 논리적으로 분리됨

공통 시트는 통합, 전용 시트는 분리라는 원칙이 명확해야 함

다운로드 결과물의 구조를 피그마상에서 이해 가능해야 함

9. 상세 페이지와의 관계

회사 Row의 상세보기 버튼은 그대로 유지하되,
그 상세 페이지는 회사 단건 데이터 확인용이다.

반면 공급망 카드 헤더의 엑셀 다운로드 버튼은
공급망 전체 통합 파일 미리보기 및 다운로드용이다.

즉 두 기능을 명확히 구분하라.

상세보기 = 회사 단건

엑셀 다운로드 = 공급망 전체 통합

10. 엑셀 업로드 버튼

엑셀 업로드 버튼 클릭 시에도
같은 방식으로 업로드 대상 템플릿 미리보기 모달 또는 업로드 팝업을 설계할 수 있도록 버튼만 함께 배치한다.

현 단계에서는 다운로드 미리보기 모달 설계를 우선한다.
