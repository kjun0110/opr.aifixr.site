AIFIX – 데이터 관리 탭 (자사 + 협력사 통합 구조 재설계)

너는 제조업 공급망·PCF 통합 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 데이터 관리 화면은:

자사 데이터만 직무 관점 전환 구조로 설계되어 있고

기존에 존재하던 “협력사 데이터 관리” 세부탭이 사라진 상태이다.

이는 잘못된 구조다.

AIFIX는:

자사 + 협력사 데이터를 하나의 업무 흐름 안에서 관리해야 한다.

✅ 1️⃣ 상위 구조 재정의
기존 상단 구조 유지
[ 조회 조건 ]
[ 구매 관점 | PCF 관점 ]

이 관점 토글은 유지한다.

🔷 하지만 탭 구조는 다음과 같이 변경하라

기존:

[ 조직/사업장 관리 ]
[ 자재/BOM 관리 ]
[ 설비/공정 구조 관리 ]
[ 계약/증빙 관리 ]

이를 다음 구조로 확장한다:

[ 자사 데이터 관리 ]
[ 협력사 데이터 관리 ]

이 두 개는 항상 존재해야 한다.

✅ 4️⃣ 협력사 데이터 상세보기 모달

행 클릭 시 상세 패널:

✅ 5️⃣ 업무 흐름 반영 (중요)

화면 상단에 진행 상태 표시 추가:

1️⃣ 공급망 구조 확정 (구매)
2️⃣ 데이터 수집 완료
3️⃣ PCF 검증
4️⃣ 승인

구매가 1단계를 완료해야
PCF 단계 활성화되도록 UI 설계.

✅ 6️⃣ 권한 분리

구매 직무:

공급망 구조 수정 가능

협력사 정보 수정 요청 가능

승인 전 상태 관리

PCF 직무:

계산 검토

DQR 검토

승인 여부 결정

✅ 7️⃣ UI 목표

이 화면은:

구매 담당자는 “공급망 통제 시스템”처럼 느껴야 하고

PCF 담당자는 “탄소 계산 검증 도구”처럼 느껴야 한다

하지만 실제 데이터는 하나의 통합 제출 템플릿을 사용한다.

🎯 최종 구조 요약

데이터 관리

├─ 자사 데이터 관리
│ ├─ 구매 관점 탭
│ └─ PCF 관점 탭
│
└─ 협력사 데이터 관리
├─ 공급망 구조 관리 (구매 중심)
├─ 협력사 제출 데이터 (관점 전환 가능)
└─ 검토/승인 현황

이 구조로 전면 재설계하라.
