AIFIX – 데이터 조회 / 데이터 관리 탭 구조 정리

너는 제조업 공급망 ESG/PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 상단 네비게이션에는 다음 두 개의 탭이 존재한다.

데이터 조회

데이터 관리

하지만 실제 사용자 업무 흐름을 고려했을 때
이 두 탭을 분리하는 것은 UX 측면에서 비효율적이다.

사용자는 데이터를 조회한 뒤 수정하려면

데이터 조회 → 데이터 관리 탭 이동 → 동일 조건 재조회

과정을 거쳐야 하기 때문이다.

따라서 데이터 조회 탭만 남기고 데이터 관리 탭은 삭제하는 방향으로 UI를 재구성하라.

별도의 데이터 관리 화면은 필요 없다.

4️⃣ 데이터 관리 탭 삭제

상단 네비게이션에 존재하는

데이터 관리 탭은 완전히 삭제한다.

주의 사항

다음 작업을 수행하라.

상단 메뉴에서 "데이터 관리" 제거

해당 화면 Frame 삭제

해당 IA 노드 제거

5️⃣ 데이터 관리 화면 요소 사용 금지

현재 존재하는 데이터 관리 화면의

다음 UI는 사용하지 않는다.

예

이 화면의 디자인은 데이터 조회 탭으로 가져오지 않는다.

7️⃣ UX 목표

최종 구조에서는

사용자가

조회 → 수정

을 하나의 화면에서 수행할 수 있어야 한다.

Workflow

데이터 조회
→ 결과 확인
→ 상세보기
→ 수정

즉

조회와 관리가 하나의 화면에서 이루어지는 구조로 설계한다.
