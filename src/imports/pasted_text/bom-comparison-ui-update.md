피그마 수정 프롬프트 (그대로 전달용)

현재 공급망 그래프 화면에서 BOM 비교 버튼을 눌러도 UI 변화가 거의 없어 사용자가 비교 기능을 직관적으로 이해하기 어렵다.

따라서 BOM 정보를 그래프 내부에서 표현하는 방식이 아니라, 그래프 영역 외부에서 명확히 보여주는 구조로 UI를 수정한다.

다음 기준으로 화면을 수정한다.

1. BOM 비교 버튼 역할 변경

현재

BOM 비교 버튼

을 누르면 그래프 내부 노드 강조만 이루어지는데,
이 방식은 사용자가 어떤 BOM이 선택되었는지 직관적으로 이해하기 어렵다.

따라서 BOM 비교 버튼 동작을 다음 방식으로 수정한다.

세부제품 선택
↓
BOM 비교 버튼 클릭
↓
선택한 BOM 정보 카드 생성
2. BOM 정보 카드 UI 추가

그래프 영역 상단 또는 우측 영역에 BOM 정보 카드를 표시한다.

예시 구조

[BOM: A']

BOM Code : SCV-2026-03  
Product : 제품A  
Tier1 Supplier : 3  
Tier2 Supplier : 5  
Tier3 Supplier : 8

카드는 다음 형태로 디자인한다.

Width : 220px
Height : auto
Card radius : 12px
Background : white
Shadow : subtle
3. 여러 BOM 비교 가능 구조

사용자가 여러 세부제품을 선택하여 BOM 비교를 할 수 있도록 한다.

예

[BOM: A']
[BOM: A'']
[BOM: A''']

이 카드들이 그래프 영역 외부에 나열되도록 한다.

레이아웃 예시

공급망 그래프
────────────────────────

[BOM A']   [BOM A'']   [BOM A''']
4. 그래프와 BOM 카드 연결 UX

사용자가 특정 BOM 카드를 클릭하면

그래프에서

해당 BOM에 포함된 노드 → 강조
해당 BOM과 무관한 노드 → opacity 30%

가 적용된다.

5. 그래프 내부 UI 단순화

그래프 노드 내부에서는 BOM 정보를 표시하지 않는다.

노드는 다음 정보만 유지한다.

Tier
회사명
국가
공급 품목
CO₂ 기여도

그래프는 공급망 구조 시각화 역할만 담당한다.

6. 선택 상태 표시

현재 선택된 BOM 카드는 다음 스타일을 적용한다.

border : Primary color (#5B3BFA)
background : #F3F0FF

비선택 카드

border : #E5E7EB
background : white
7. BOM 카드 삭제 기능

각 BOM 카드 우측 상단에

✕ remove

버튼을 추가하여 비교 목록에서 제거할 수 있도록 한다.

기대 UX 효과

이 변경으로 다음 문제가 해결된다.

BOM 비교 기능 가시성 향상
그래프 정보 과부하 감소
사용자 인지성 개선

그래프는 공급망 구조,
BOM 카드는 제품 구성 정보를 담당하도록 역할을 분리한다.

추가로 하나만 UX 팁

지금 그래프 UI는 거의 완성 단계인데
딱 하나만 더 하면 발표용 완성도가 확 올라가.

BOM 카드에 이것 추가하면 좋아.

PCF total
예: 52 kgCO₂e

그러면 발표할 때 이렇게 말할 수 있음.

"제품 BOM별로 공급망 구조와 탄소 기여도를 동시에 비교할 수 있습니다."

이거 면접관이 엄청 좋아하는 포인트야.
