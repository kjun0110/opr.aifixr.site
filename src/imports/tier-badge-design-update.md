AIFIX – 데이터 조회 화면 / Tier 텍스트 가독성 오류 수정

너는 AIFIX의 시니어 프로덕트 디자이너다.
현재 [데이터 조회] 화면의 Tier 표시 UI에 가독성 문제가 발생했다.

🎯 문제 상황

조회 결과 테이블의 좌측 “Tier” 컬럼

“공급망 MAP 보기” 버튼 클릭 후 열리는 MAP 패널의 노드 상단 Tier 뱃지

위 두 영역에서:

Tier 1 / Tier 2 / Tier 3 텍스트가

보라색 배경 안에 존재하나

글자가 거의 보이지 않음 (대비 부족 문제)

즉, 텍스트 컬러와 배경 대비가 맞지 않아 접근성 문제가 발생한 상태

Tier 뱃지의 가독성 확보

AIFIX 디자인 시스템 유지

2️⃣ Tier 뱃지 디자인 수정 지시
