AIFIX – 데이터 관리 탭 회사 상세 페이지 + 공급망 Excel 다운로드 설계

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 [데이터 관리] 탭에서 조회된 공급망 리스트에서
각 회사 Row에 상세보기 버튼이 존재한다.

이 버튼을 클릭하면 팝업이 아니라 새로운 페이지(depth page) 로 이동하도록 UI를 설계하라.

중요 조건:

현재 조회 화면에는 구매 관점 / PCF 관점 토글이 존재한다

그러나 회사 상세 페이지는 두 관점과 무관하게 동일한 화면을 보여야 한다

즉 직무별 상세 페이지 분리는 없다

정리

조회 화면 → 직무 관점 존재
상세 페이지 → 직무 관점 없음
