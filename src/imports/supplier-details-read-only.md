AIFIX – 협력사 상세보기 화면 재설계 (원청 조회 전용)

너는 제조업 공급망 ESG / PCF 관리 SaaS AIFIX의 시니어 프로덕트 디자이너다.

현재 데이터 관리 화면에서 협력사 상세보기를 누르면
협력사 전용 입력 화면 구조가 그대로 노출되고 있다.

그러나 현재 화면은 원청 시스템이다.

따라서 원청은

협력사 데이터를 조회만 가능

직접 수정 불가

수정 요청만 가능

하도록 UI를 수정하라.
